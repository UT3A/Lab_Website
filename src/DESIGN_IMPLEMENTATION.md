# FAIM Lab 網站設計實現文檔

## 📋 專案概述

本專案根據詳細的 Figma 設計指令，完成了一個融合極簡高級科技感的 AI 實驗室網站。採用現代化的深色科技美學風格，注重細節與用戶體驗。

---

## 🎨 視覺系統

### 配色方案

**主色調 - 深色基底：**
- Background Primary: `#020617` (slate-950) - 主背景
- Background Secondary: `#0f172a` (slate-900) - 卡片背景
- Background Tertiary: `#1e293b` (slate-800) - 次要元素

**科技藍漸變 - 品牌色：**
- Cyan Primary: `#22d3ee` (cyan-400) - 主要強調色
- Blue Primary: `#2563eb` (blue-600) - 次要強調色
- Purple Accent: `#a855f7` (purple-500) - 點綴色

**文字層級：**
- Text Primary: `#ffffff` - 主標題
- Text Secondary: `#cbd5e1` (gray-300) - 內文
- Text Tertiary: `#94a3b8` (gray-400) - 輔助文字

### 字體系統

- 西文: Inter / SF Pro Display
- 中文: Noto Sans TC / 思源黑體
- 統一使用 Tailwind 預設字級系統

### 視覺效果

1. **玻璃擬態 (Glassmorphism)**
   - 背景: `slate-900/50`
   - Backdrop Blur: `backdrop-blur-xl` (24px)
   - 邊框: `border-cyan-500/20`

2. **發光效果 (Glow)**
   - 使用 `shadow-cyan-500/30` ~ `shadow-cyan-500/50`
   - 結合 `hover:shadow` 實現互動

3. **漸變動畫**
   - 標題文字使用 180deg 線性漸變
   - 按鈕背景使用 45deg 漸變
   - 配合 Motion 實現平滑過渡

## 🎬 動畫與互動

### 進場動畫
- 所有區塊使用 `useInView` 觸發
- 初始: `opacity: 0, y: 30`
- 動畫: `opacity: 1, y: 0`
- 持續時間: 600ms
- 延遲: 依序 +100~150ms

### 懸停效果

**按鈕：**
- Scale: 1.05
- Shadow 增強
- 圖標平移

**卡片：**
- TranslateY: -4px ~ -8px
- 邊框顏色加深
- 光暈顯示

**圖片：**
- Scale: 1.02 ~ 1.1
- Filter: brightness 增強
- 平滑過渡

### 微互動
- 導航底線: 寬度 0% → 100%
- 圖標旋轉: rotate(360deg)
- 按鈕 ripple 效果
- 表單驗證提示

## 🔧 技術實現

### 核心技術棧
- **框架**: React + TypeScript
- **樣式**: Tailwind CSS v4.0
- **動畫**: Motion (Framer Motion)
- **圖標**: Lucide React
- **構建**: Vite

### 組件架構
```
components/
├── Navigation.tsx          # 導航欄
├── HeroSection.tsx         # 主視覺
├── PhilosophySection.tsx   # 理念展示
├── ResearchSection.tsx     # 研究領域
├── AchievementsSection.tsx # 成果展示
├── TeamSection.tsx         # 團隊成員
├── JoinSection.tsx         # 聯繫我們
├── Footer.tsx              # 頁腳
├── ChatBot.tsx             # AI 聊天機器人
├── AnimatedBackground.tsx  # 動態背景
├── GridBackground.tsx      # 網格背景
└── FloatingElements.tsx    # 浮動元素
```

### 性能優化
- 使用 `useInView` 延遲動畫
- 圖片使用 `ImageWithFallback` 組件
- 動畫使用 GPU 加速 (transform, opacity)
- 分層渲染避免重繪
